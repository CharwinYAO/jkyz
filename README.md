# 健康驿站抢票
semi-automated ticket robing system for jiankangyizhan

# 功能
1. 半自动化的抢票系统

2. 设置时间，到点加载预约时间页面。

3. 检测到有橙色可预约日期时，自动载入。（默认从第7天检测到第2天）

4. 检测到确认预约界面时，自动打开验证码。

# 注意

由于确认预约页面的验证码更新以后，无法被电脑轻易破解，所以在验证码弹出以后，只能手动点击，完成后手动确认预约。

成功与否主要和网速有关，网页都加载不了的话，什么软件都白搭。
